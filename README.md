# 🌱 Agricultural Irrigation Data Analysis Dashboard

An interactive **Power BI dashboard project** that analyzes agricultural retail price data across different regions, products, and time periods. This project provides insights into pricing trends, regional variations, and product-wise performance using real-world datasets.

---

## 📊 Project Overview

This project focuses on analyzing **daily retail prices of essential agricultural commodities** such as:

- Rice  
- Onion  
- Cabbage  
- Brinjal  
- Groundnut Oil (Packed)

The datasets were collected from the **Open Government Data (OGD) Platform India** and combined to build an insightful dashboard.

---

## 🎯 Objectives

- Analyze **total sales and price distribution** across regions  
- Identify **price trends over time**  
- Compare **product-wise performance**  
- Visualize **regional variations using maps**  
- Build an **interactive and user-friendly dashboard**

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Dashboard creation & data visualization  
- **Microsoft Excel / CSV** – Data source  
- **Power Query** – Data cleaning & transformation  
- **DAX** – Calculations & KPIs  

---

## 📈 Dashboard Features

- 📌 KPI Cards (Total Sales, Avg Price, Max Price, Min Price)  
- 📌 Product-wise Analysis (Pie Chart)  
- 📌 Region-wise Sales (Bar Chart)  
- 📌 Time-based Trends (Line Chart)  
- 📌 Map Visualization (Sales Distribution by Region)  
- 📌 Interactive Filters (Slicers for Region & Product)  

---
